# UW Kendo Web UI
### [Heroku Deployed View](https://uw-kendo-web.herokuapp.com/)
### Developer Usage:
   - For first-time initialization:
     - `yarn install`
     - `yarn start`
     - run on `http://localhost:3000`
   - For test:
     - `yarn test`
   - DON'T PUSH `build/`, which will block heroku auto deployment.
